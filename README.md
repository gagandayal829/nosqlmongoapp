## Commands

Clone this repo

```
git clone https://github.com/gagandayal829/nosqlmongoapp.git
```

Initializes the app

```
npm init
```

Install the dependencies

```
npm install express body-parser mongoose --save
```

Install nodemon globally

```
npm install nodemon -g
```

Start the app

```
nodemon server.js
```
